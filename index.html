The plan is to make some pointer animation things just like we use to have on the Internet in the good old days

possible options:
  • When the mouse moves a trail of icons follows it. They can fade? They can stay?
  • When the mouse moves it outputs a trail of text
  • Can we make the cursor leave sparks behind it?
  • How about an image or animation that follows the cursor

This one is an image that follows the cursor. Just whatever the defaults are. The default image is of course the head of the legendary Macho Man Randy Savage.

If I get around to it I will standardise the speed it travels at. 
          $(".following-image-row-1").StartShenanigans("followingImage")











This one is the same as the above, an image that follows the cursor, but you can change image. I have used an image that could be a ufo or it could be a donut. 

          $(".following-image-row-2").StartShenanigans("followingImage", { image : "images/ufo-or-donut.png" })











This is an image that flies at the cursor. The image needs to start facing to the left. It can fly over the whole page, but is only listening to mouse events in the container it is put in (e.g. the below is '.flying-image-row-1'), so if the mouse moves outside that container then it will stop following the mouse and instead go to the last known position 

          $('.flying-image-row-1').StartShenanigans("flyingImage");











This one will cause the cursor to leave a trail of text behind it. Just whatever the defaults are. 
          $(".text-trails-row-1").StartShenanigans("textTrail");











This is another text trail, give it some styling, set the text, only output it just the once, don't let it fade so it stays forever 
          $(".text-trails-row-2").StartShenanigans("textTrail", {
            "style" : "color: #830000; font-size: 1.1em;",
            "fadeText" : false,
            "repeatText" : false,
            "textToDisplay" : "Welcome to the grungle.   ",
          });











This is the icon trail. All you have to do is pass in an image like this spooky ghost 

                  $('.default-row').StartShenanigans( 'iconTrail',
                    { 
                      'icon' : 'images/mystery.png'
                    }
                  );

Also icon trail. For this area we are going to use the fontawesome library to provide the image - our default is the skull 

                $('.first-row').StartShenanigans( 'iconTrail',
                  {
                    'useFontAwesome' : true, // use fontawesome icons - the default is 'skull'
                  }
                );

Also icon trail, set some options. 

                $('.second-row').StartShenanigans( 'iconTrail',
                { 
                  'useFontAwesome' : true,
                  'icon' : 'ghost',         // pick a different icon 
                  'iconColour' : '#fff', // set the colour of the icon
                  'iconScale' : 0.75,          // smaller ghost is scarier, harder to see!
                  'travelDistanceForNewIcon' : 10,  // make them come closer together
                  'makeIconDelay' : 5
                }
              );

Also icon trail, set some different options. 


                $('.third-row').StartShenanigans( 'iconTrail',
                { 
                  'useFontAwesome' : true,
                  'icon' : 'bomb',
                  'travelDistanceForNewIcon' : 15,  // make them come closer together
                  'opacityIncrement': 0.05, // really slow fade
                  'fadeDelay': 100, // really slow fade
                  'makeIconDelay' : 20
                }
              );

Icon trail, don't fade the icons 

                $('.forth-row').StartShenanigans('iconTrail',
                { 
                  'useFontAwesome' : true,
                  'icon' : 'shoe-prints',
                  'iconColour' : '#6fa36e',
                  'fadeIconEnabled' : false, // don't fade out the icons
                }
              );

This one is the sparks animation. Whatever the default is. 

                $('.fifth-row').StartShenanigans( 'sparks');










This will try to stick the sparks to the cursor 

                $('.fifthish-row').StartShenanigans( 'sparks', {
                  "sparkSticksToMouse" : true,
                  "percentChangeOfSparkOnMove" : 10
                });










This one is HARDCORE sparks. 

                $('.sixth-row').StartShenanigans( 'sparks', {
                  "percentChangeOfSparkOnMove" : 100

Sparks again, but this one does lots of colours. 

                $('.seventh-row').StartShenanigans( 'sparks', {
